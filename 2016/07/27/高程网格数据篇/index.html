高程网格数据篇

发表于 2016-07-27

GSD (Swedish DEM Grid) Files
GSD格式文件用于储存瑞典DEM数据。GlobleMapper11及以上版本支持这种格式的数据。

GXF (Geosoft ASCII Grid) Files
GXF是一种用于储存高程数据的ASCII网格数据格式。GlobleMapper11及以上版本支持这种格式的数据。

Geodas Grid (GRD98) Format
这是用于储存网格数据的数字格式。虽然是为bathymetric/topographic开发的，但这个格式能应用于几乎所有的网格数据。GlobleMapper8及以上版本支持该格式。

Geosoft Binary Grid Files
Geosoft Grid是一种二进制的网格格式，用于储存高程数据。此格式同时使用Geosoft’s Oasis Montaj操作。

Global 2’ Elevation Data (ETOPO2)
一个新建的全局高程数据库（global elevation data base），以经纬度分辨率查看（gridded at latitude-longitude resolution）。ETOPO2包含了陆地数据（Global Land One-kilometer Base Elevation，GLOBE）和各种海底数据。
Global 30-arc-second Digital Elevation Data (GTOPO30)
GTOPO30数据由一些光栅数据和地形矢量资料组成。数据来源于一些美国国内组织据和国际组织，此数据与USGS DEM数据不同。
Gravsoft Grid Files
此格式是Gravsoft的套件工具，用于储存高程网格数据。Global Mapper10及以上版本可导入此格式的数据。

HF2/HFZ Files
HF2是一种用于替代压缩地形格式（uncompressed terrain formats）的压缩高度场格式（compressed heightfield format），如HFF、TER和BT。
HTF (Hydrographic Transfer Format)
HTF主要用于澳大利亚的Australian Hydrographic Service (AHS)部门，该格式目前已经成为首选的AHS探测数据（sounding data）储存和分配格式。

Hypack Matrix Files
Hypack Matrix格式一种网格格式，用于储存海洋测量的深海数据（bathymetric data）。Global Mapper11及以上版本支持该数据格式。Global Mapper 11及以上版本可以导入该格式文件。
Idrisi格式
Idrisi .rst是一种栅格数据格式，用于Clark Labs的Idrisi软件生产线。

International Bathymetric Chart of the Arctic Ocean (IBCAO) Files
IBCAO格式是用来开发数字数据库（digital data base）的，适用于现有的所有bathymetric数据。Global Mapper以netCDF格式支持这种数据。

Japanese DEM (JDEM/LEM/CSV) Format
JDEM .mem/.lem文件格式为Japanese Geographical Survey Institute提网格高程数据。Global Mapper11及以上版本支持该数据。

LizardTech MrSID MG4 Lidar
该种格式可对Lidar数据集进行超高比例的压缩储存。Global Mapper11及以上版本支持该数据。

NOAA TerrainBase Elevation Data
该数据格式支持NOAA National Geophysical Data Center里的所有数字地面模型（Digital Terrain Models）。

NTF Grid/Contour Format
NTF Grid网格文件格式用于UK Ordnance Survey储存DEM格式数据和矢量数据。Global Mapper8及以上版本支持该格式的数据。

PDS (Vicar) Files
PDS (Vicar)文件格式是二进制的网格格式，NASA通常用于储存行星数据（planetary data）。Global Mapper10.01及以上版本支持该格式的数据。

PLS-CADD XYZ
PLS-CADD XYZ文件格式用于Power Line Systems的PLS-CADD程序储存矢量数据和网格高程数据。Global Mapper9.02及以上版本支持该格式的数据。

QCT (Quick Chart) and QED Files
QCT和QED文件格式用作高度压缩的扫描地图（highly compressed scanned maps (QCD)）和高程数据文件，主要储存于类似PDA和便携GPS装置中。Global Mapper11及以上版本支持该格式的数据。

STL文件
STL是一种由3D系统产生的立体印刷（stereolithography）CAD软件。此格式可用于多种软件包。Global Mapper各版本都支持该数据格式。
